# Lanzarus 📈

**AI Trading & Finanzas — Android App**

App nativa Android con IA para análisis de mercados crypto en tiempo real, trading automático y panel de inversiones.

## Características

- 📊 Precios crypto en tiempo real (CoinGecko API)
- 🤖 Bot IA con análisis técnico (RSI, SMA, Volatilidad)
- 💼 Cartera de inversiones con P&L
- 💬 Chat con asistente financiero IA (Gemini)
- 📱 AdMob integrado (banner, intersticial bonificado, bonificado)
- 🔔 Notificaciones de alertas de mercado
- 🌙 Tema oscuro premium

## Build

El APK se genera automáticamente con cada push a `main` via GitHub Actions.

Ver [Releases](https://github.com/joanakar3dmoon/Lanzarus-/releases) para descargar el último APK.

## Instalación

1. Descarga el APK desde Releases
2. Android → Ajustes → Seguridad → Fuentes desconocidas → Activar
3. Abre el APK e instala
